# Curiosity — Quote by Clay P. Bedford

This repository contains a beautiful PNG infographic of a quote by **Clay P. Bedford** about teaching curiosity and lifelong learning.

> "You can teach a student a lesson for a day; but if you can teach him to learn by creating curiosity, he will continue the learning process as long as he lives."  
> — Clay P. Bedford

## Contents
- `index.html` — a small, elegant web page to view and share the quote.
- `quote.png` — the infographic image (drop this file into the repo).
- `README.md` — this file.

## How to use
1. Put `quote.png`, `index.html`, and `README.md` in the repository root.
2. Push to GitHub.
3. In the repository Settings → Pages enable GitHub Pages (select branch and `/root`).
4. Share the published URL (e.g. `https://yourusername.github.io/your-repo/`).
